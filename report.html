Project 1 Report

Below is a list, corresponding to the list in the twodee blog post, Project 1 - Handcrafted Site, and how I believe I have met each of those requirements
  1. Client is my cat from home, Ziggy. Client is not me
  2. Header is present on every normally visible page
  3. Site contains 6 pages, each with a purpose
  4. Site contains navigation bar leading to all pages
  5. On Android and Desktop my website displays fine, for some reason on iOS I guess it does not, however I don't have an Apple device to test it and all my mobile format testing on desktop Firefox/Google Chrome and on my own phone (Samsung Galaxy S9+) it has displayed fine.
  6. Global style is in master.css and individual pages have their own .css file
  7. All images have alt attributes
  8. Favicon is declared on each page
  9. images, styles, and scripts folders used
  10. No additional frameworks/libraries used
  11. HTML validator and CSS validator passes each page
  12. Site is on github and shared link with instructor
  13. All html/js related files are lowercase
  14. port 8443 is used
  15. site uses HTTPS
  16. Axes clearly defined and site is structured around them
  17. Two different fonts used. Not the normal sans/sans-serif fonts, but ones that I feel fit the page nicely
  18. Form used in 'Contact Me' page. The action doesn't do much, just clears out when submit button is pressed and triggers text to show
  19. Links to facebook and twitter used in Contact Me page and they point to _blank for new tab
  20. source code, in my opinion, is readable with proper camelcase/format for names and tabbing
  21. animation in header, Ziggy's head flipping every few seconds. Powered by javascript

Client Critique - Ziggy's family

Overall - very well done. Very humorous! Love the cat point of view. Matthew was disappointed Ziggy didn't have his own FB or twitter account. Desktop the website flows smoothly. Everything looked fine on Android as well. We like Ziggy's moving head on the top.

A couple of things on Apple, the schedule overlaps with everything (time overlaps with sentences) so you can't read it. On the family section, the names Zoey and Peter overlap a little onto the photo in portrait. Landscape it is fine. Also the contact page fields overlap a little with the FB & Twitter icons. Dad will send you screen shots so you can see.
